# 更新

检查 trainsh 的新版本。

## 何时使用

- 查看当前安装版本是否落后。

## 命令

```bash
train update --help
```

## CLI 帮助输出

```text
[--help] [--check]

Check for updates and automatically upgrade tmux-trainsh.

Options:
  --check    Only check for updates, don't install

Examples:
  train update          # Check and install update
  train update --check  # Only check, show update command
```
